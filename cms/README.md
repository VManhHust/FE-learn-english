# Learn English CMS

Admin interface built with React-admin, TypeScript, Vite and Material UI.

## Development

Start the Spring Boot API on port `8080`, then:

```bash
npm install
npm run dev
```

Open `http://localhost:5173` and sign in with an account whose role is `ADMIN`.

Vite proxies `/api` requests to `http://localhost:8080` during development.
